# fantasy-football

# Requirements

- Node v10.13.0: https://nodejs.org/en/download/
- PostgreSQL v10.5

## Running the project:
1. Start pgAdmin4 locally to start the database.
1. Load SQL scripts into running database.
1. Install all the things:
  `cd fantasy football && npm install && npm run pre`
1. Run the app:
  `npm run start`
